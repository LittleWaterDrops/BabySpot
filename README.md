# 🛎️ 프로젝트 소개

## 1. 서비스명 및 개요

### 서비스명

- 서비스명 : 베이비 스팟

### 개요

- 서울시 영유아 친화 가게 추천 서비

## 2. 주요 문제 해결

### **1)** 아이 동반 고객을 위한 **편의 용품/서비스 정보 부족**

- 현재 음식점 정보에는 **유아용 의자, 놀이 공간, 키즈 메뉴 여부 등 필수 정보 부족**
- 부모들이 원하는 **유아 친화적인 음식점을 찾는 것이 어려움**

### **2)** 아이와 함께 외식 시, **적절한 장소 정보를 찾기 어려움**

- 일반적인 지도 앱(네이버, 구글)에서는 **가족 친화적인 장소를 필터링하기 어려움**
- **실제 방문자 리뷰 기반의 신뢰도 높은 데이터 부족**

## 3. 서비스 주요 기능

### 1) 영유아를 데리고 가기 좋은 식당에 대한 정보 제공

- GPS 기반 가까운 식당 중 영유아 친화적 음식점 리스트 제공 (서울시 오케이존 데이터 사용)
- 필터링 기능 제공 (6인 이상 테이블, 유아 의자, 유아 식기, 놀이방 등)
- 해당 식당의 리뷰(네이버 플레이스, 카페, 블로그 글or 리뷰)를 크롤링하여 어린이 메뉴 및 아이를 동반한 가족이 작성한 리뷰의 키워드 추출

### **2) 자체 리뷰 서비스**

- 유저가 직접 별점 및 시설 만족도 평가 가능 → 서비스 자체적인 리뷰 시스템을 이용하여 아이와 동반된 구체적이고 다양한 리뷰들을 제공
- 필터링 기능을 통해 자신의 가족과 비슷한 가족 구성원의 리뷰 제공

### **3) 내 자녀 정보를 기반으로 주변 식당 추천**

- 사용자가 방문하고자 하는 아이의 정보(인원수, 나이)를 통해 우리 자체적 리뷰를 분석하여 맞춤형 가게 추천

## 4. 서비스 흐름

1. **사용자 위치 기반 → 주변 유아 친화 음식점 추천**
2. **필터링 옵션 제공 (놀이방 여부, 키즈 메뉴 여부 등)**
3. **사용자 리뷰 데이터 분석 → 신뢰성 높은 추천 제공**

## 5. 기대 효과

### **1) 부모들이 쉽게 유아 친화적인 음식점을 찾을 수 있음**

- "우리 아이와 가기 좋은 음식점 어디 있지?"에 대한 고민 해결
- 음식점 방문 전 **유아 편의 용품 여부를 쉽게 확인 가능**

### **2) 데이터 기반으로 신뢰도 높은 추천 제공**

- 단순한 광고 기반 추천이 아닌, **실제 방문자 리뷰 및 데이터 분석을 통한 추천**

# 📄 기능 명세

### **1. 유아 친화 음식점 추천**

- **GPS 기반 현재 위치 주변 추천**
- **서울시 오케이존 데이터 활용하여 유아 친화 가게 필터링**

### **2. 맞춤형 필터링 기능**

- **유아용 의자, 놀이방, 기저귀 교환대 등 필터 적용 가능**
- **6인 이상 테이블 여부, 키즈 메뉴 제공 여부 등 추가 필터 제공**

### **3. 유저 리뷰 및 데이터 분석**

- **크롤링을 통한 "유아 친화적" 키워드 자동 추출 → 매장 상세정보 반영**
- **서비스 자체 리뷰 시스템 운영 (아이 동반 고객 중심 리뷰 제공)**
- **유저가 직접 시설 만족도 평가 및 리뷰 작성 가능**

### **4. 사용자 기반 장소 추천**

- **사용자가 입력한 자녀의 정보를 토대로 자체 리뷰데이터를 분석해서 가게를 추천**

### **5. UI/UX 최적화**

- **React Native 기반 모바일 최적 UI 제공**
- **직관적인 지도 기반 검색 기능**
