## 1. 개발 트랜드

### Target

- 취업 준비중인 주니어 개발자

### Pain Point

- 기술 스택을 선정하고 이해에 대한 어려움이 있음
- 취업 시장에서 요구하는 기술이 어떤 것이 있는지 한눈에 파악하기 어려움

### Service

- 주니어 개발자를 위한 기술 트렌드 정보 제공 서비스

### Main

- 채용 공고를 기반으로 한 메인 기술 트렌드
- 기업별 채용공고 분석
- 카테고리별 선호 기술

### Sub

- 최근 10년간의 기술 트렌드 변화
- 개발자 mbti로 초기 단계에서 설문조사

## 2. 인기 유튜버의 비결

### Target

- 인기있는 유튜브 영상을 제작하고 싶은 사람

### Pain Point

- 어떻게 해야 유튜브에서 좋은 호응을 얻을 수 있는지 모름

### Service

- 인기 유튜버의 비결을 분석하는 서비스

### Main

- 인기 유튜버의 영상 패턴을 분석
- 구독자 유치 및 유지 비결을 제공
- 본인과 인기 유튜버를 비교

### Sub

- 숏폼 트렌드 분석

## 3. 제주도 여행 계획

### Target

- 제주도 여행 계획을 쉽게 짜고 싶은 사람

### Pain Point

- 자주 가는 사람들도 어디를 가야 하는지 잘 모름
- 많은 사람들이 다니던 관광지만 가게 됨

### Service

- 사용자 선호 기반 제주도 여행 계획 제공 서비스

### Main

- 사용자가 정한 기준에 기반, 여행 계획 랜덤 생성
  - 카테고리는 숙소, 먹거리, 관광지가 있으며 추천 받을 카테고리의 개수는 고정
  - 선입력을 통해 숙소나 음식점을 출발지로 설정
  - 자차를 기반으로 경로를 제공

### Sub

- 식당이 폐업이거나 휴무일 경우, 주변 위치를 기반으로 재추천

## 4. 과일 정보 제공

## 🚫 개발자 트렌드

- 기획과 정확히 똑같은 서비스가 이미 존재
- 분석할 만큼 많은 데이터가 필요한지 모르겠고, 답이 정해져 있는 느낌이 강함
- 규모별, 업종별로 나뉘지만 큰 변화는 없을 것
- 1년 단위로 키워드를 보여주는 사이트가 존재하지만 그 정도 텀으로는 서비스가 어려울 것. 빈번히 사용할 서비스가 필요해보임

## ✅ 인기 유튜버의 비결

- 제목은 어그로가 끌렸는데, 풀어가는 내용은 다큐인 느낌
- 인기 유튜버의 인기 추이를 파악할 수 있으니 좋아 보임. 다만 우리가 어떤 요소까지 접근이 가능할지 궁금함.
- 영상이 나오면 스크립트가 제공될 것임. 댓글처럼 스크립트에 접근하면 좀 더 납득 가능한 결과를 도출할 수 있지 않을까?
- 방향성 제시가 좋은 것 같음. 하지만 디테일하게 몇 분동안 해서 인기가 있어~ 는 어색해보임
- 해당 영상의 시간별 조회수를 따로 뽑을 수 있는지에 따라 달라질 것 같음
- 아이디어가 좋고 이후에 어떻게 풀어나갈지가 중요해보임
- 빅데이터에 어울려보임

## 🚫 제주도 랜덤 여행

- 단순 추천 서비스 아닌가 싶음
- 어느 방향으로 갈지 감이 안 옴
- 빅데이터와 연관성이 없어보임. 데이터만 넣고 랜덤으로 돌리는 것 같음
- 상위 지점에 점수를 매겨 판단한다고 했는데, 점수를 매긴 순간, 랜덤이 아님. 결국 같은 결과만 나오지 않을까 생각함. 단순 순위 매기기지 추천 서비스와도 거리가 멀어보임.
- 너무 전형적이라 우리가 고민할 부분이 적어보임.
- 여행에 재미요소를 더하거나, 개인화 되어있거나. 요소를 더 추가해야 할 듯
- 살면서 몇 번 안 가는데 개인화가 가능할지도 의문.
- 추천 퀄리티가 높지 않음. 이후에 검색해본 내역을 통해 데이터가 쌓이는 것
- 해당 주제를 많이 진행하긴 했음, 다만 지속적인 크롤링이 필요함

## 🚫 과일 정보 제공

- 가격이 다르다고 소비량에 큰 차이가 없다고 생각
- 이미 해당 서비스가 존재
- 정확히 어떤 데이터로 어떤 것을 제공한다는 것인지 명확하지 않아보임.
- 굳이 대상이 일반인일 필요가 있는지도 모르겠다. 빅데이터와의 연관성도 흐릿한 것 같음.
- 이 서비스의 목적성에 대한 고민이 더 필요해 보임

# 25.03.06 새 기획을 위한 Keyword

### 유튜브는 충분한 부분에서 대화를 나누었다. 아직은 시간이 없는 것이 아니니 잠시 넣어두었다가, 좋은 기획이 없으면 고려해보자!

<aside>

한 사람 씩 돌아가면서 아무 키워드를 생각나는 대로 던진다.

주어진 시간 동안 해당 키워드를 보고 생각나는 얘기를 나누어 본다.

한 사이클이 돌았다면, 더 구체화가 가능할 키워드를 남기고 모두 지운다.

</aside>

## 1. 배달

- 시장에서 먹긴 함
- 나도모르게 지출이 커짐
- 통신 관련해서 얼마나 배달에 돈을 쓰고 있는지 나오는 데이터가 있음
- 물은 제한 수량이 존재함(무게)
- 마트 물품도 배달해서 신기했음
- 음식 시키기보다 요즘은 장보는 용도로 많이 사용함
- 쓰레기가 많이 생김
- 내 배달 음식 성향을 알 수 있으면 좋겠음
- 과소비를 많이하게 됨
- 쿠팡만큼 과대포장 많이 하는 곳이 없음
  - 이유가 뭘까? -> 빠른 일처리를 위해 보관해둔 그대로 낱개로 배달
  - 수요 예측이 그래서 중요함
  - 오배송되면 그 누구도 책임을 안 짐

## 2. 전자기기

- 전자기기는 삼성
- 전자기기에 관심이 있으면 가격대가 있기 때문에, 하루종일 가격 변동을 보고 있음
  - 공식 홈피가 아닌, 쿠팡같은 곳
- 성능, 구성품, 스펙 등을 많이 봄
  - 구매 타협점에 개인차가 있을텐데 그 기준점을 모르겠다
  - 가격대비 쓸만한가??
  - 개인적으로 판단할 때를 말한다. 판단 기준이 뭘까
- 나는 스펙을 온전히 활용하지 못함. 항상 과스펙
  - 내 스스로 스펙이 적당한지 알기가 힘듦
  - 보통 가격으로 판단
    - 어느정도 가격일때, 사람들이 타협을 하고, 본인 기준에 맞다고 생각할까?
- 원하는 니즈에 맞는 상품 스펙을 모아둘 수는 없을까?
  - 가격, 브랜드 등등

## 3. 햄버거 가게

- 잘 되는 곳들의 입점 전략을 분석하면 어떨까
- 어느 정도가 걸어서 가기에 적절한가
- 햄버거 지수를 사용해보자
  - 서울이 아닌 시골 사람들은 햄버거 가게 종류로 도시의 파워를 척도로 함
- 제주도에는 버거킹이 없었다
  - 시골에는 롯데리아
  - 읍내로 나가면 롯데리아 + 맥날
- 접근성 차이가 크다
- 스타필드도 광주는 없어서 들어간다 안 간다 말이 많음

## 4. 유동인구

- 기념일 때 신촌, 명동 이런 곳을 갈 때, 지하철을 못 나온 경험이 상당수
  - 눈치싸움
- 왜 크리스마스인데 PC방에 사람이 많아?
- 갑작스러운 시위
- 인구가 쏠리는 이유가 뭘까?
  - 왜 브랜드 상가가 들어오지 않다가 갑자기 생길까?
    - 교통이 중요해 보임
  - 왜 시골이 아닌 수도권에만 모여있을까?
    - 인프라가 중요하다고 생각함
  - 이거에 대한 연계로 이런 지역에 이런 업종의 가게를 차리면 좋을 것 같다
- 원초적인 요인?
  - 일자리
    - 근처에 일자리가 있어야 일단 사람이 모이는 듯
    - 일자리가 있으니 교통편이 발달
    - 교통편으로 인프라가 발달
    - 단지가 크지 않다면 그렇게 큰 요인이 될까?
